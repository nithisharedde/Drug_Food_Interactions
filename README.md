# Drug-Food Interaction and Taste Prediction

## Project Description
This project predicts potential drug-food interactions and drug taste using machine learning and molecular fingerprints generated with RDKit.

## Technologies Used
- Python
- Streamlit
- RDKit
- Scikit-learn
- Pandas
- NumPy

## Features
- Drug-Food Interaction Prediction
- Taste Prediction
- Streamlit Web Interface
- Molecular Fingerprint Generation

## How to Run
1. Install requirements:
   pip install -r requirements.txt

2. Run the application:
   streamlit run app.py

## Project Structure
- app.py
- interaction_model.pkl
- taste_model.pkl
- drug_smiles.csv
- food_smiles.csv
- requirements.txt
